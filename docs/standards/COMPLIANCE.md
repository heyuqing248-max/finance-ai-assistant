# Compliance Standard / 合规规范

## Positioning / 定位
The product provides financial information organization, AI analysis, and educational-style reference. It must not present itself as a licensed investment adviser unless the business later obtains required permissions.

本产品提供财经信息整理、AI 分析和教育参考，不应在未取得资质前包装成持牌投资顾问服务。

## Required Disclaimer / 必须出现的免责声明
Recommended Chinese text:

本内容由模型基于公开信息和市场数据生成，仅供学习和研究参考，不构成任何投资建议、买卖推荐或收益承诺。市场有风险，投资需谨慎，用户应结合自身情况独立判断。

## Prohibited Claims / 禁止表达
- 保证收益。
- 保证预测准确。
- 承诺规避亏损。
- 要求用户必须买入或卖出。
- 暗示平台替用户承担投资责任。

## User Data / 用户数据
- Only request data needed for analysis.
- Portfolio data must be optional.
- Explain why optional data improves analysis.
- Protect login, watchlist, preference, reminder, and analysis-history data.
- Notification permissions must be user-controlled.

## Public Product Risk / 面向社会的风险
Because the webpage/PWA is public-facing, product copy and AI output must always distinguish:
- Facts
- Model estimates
- Opinions or references
- Risks and uncertainties

## Review Gate / 审核关口
Before releasing features involving AI investment analysis:
- Check wording against this file.
- Confirm all probability numbers are labeled as model references.
- Confirm disclaimer appears in analysis output or nearby UI.
