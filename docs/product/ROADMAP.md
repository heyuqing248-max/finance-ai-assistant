# Roadmap / 开发路线图

## Phase 0: Project Setup / 项目启动
- 创建项目文件夹、标准文档和开发日志。
- 建立项目记忆文件。
- 创建每日自动日志汇总任务。

## Phase 1: Product Definition / 产品定义
- 完成 PRD。
- 明确首页、市场页、自选股页、股票详情页、AI 分析页、设置页。
- 把 MVP 功能拆成小任务，避免一次做太多。

## Phase 2: Technical Design / 技术设计
- 确定 Web/PWA 前端框架。
- 确定后端、数据库、认证、AI 服务和数据源接入方式。
- 设计新闻聚合、股票搜索、自选股、提醒、AI 分析的数据流。

## Phase 3: Prototype / 原型设计
- 先做低保真页面。
- 验证中文 UI、极简展示、量化概率、风险提示和术语解释按钮。
- 在实现真实数据前，可使用静态样例数据验证体验。

## Phase 4: MVP Development / MVP 开发
- 实现注册登录。
- 实现股票搜索和自选股。
- 实现 A 股、港股、美股新闻分类。
- 实现 AI 新闻总结和量化分析展示。
- 实现专业名词解释按钮。
- 实现网页内提醒。

## Phase 5: Stability and Safety / 稳定安全
- 增加错误处理、空状态、加载状态。
- 增加用户隐私保护、权限说明和免责声明。
- 增加基础测试和关键流程回归测试。

## Phase 6: Expansion / 后续扩展
- 增加更多市场和资产类别。
- 增加邮件、短信、微信、Telegram 推送。
- 研究券商或行情网页/API 联动。
- 研究未来商业化方案。
