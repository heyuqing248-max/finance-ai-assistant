## Current Decisions / 当前决策
- Version 1 is now a browser webpage / Web/PWA, not a native iOS, Android, Windows, or macOS app.
- For local development, prefer `node scripts/full-dev-server.mjs`; it starts both the webpage server and the mock backend together.
- When the webpage opens from `127.0.0.1:4173` or `localhost:4173`, it should automatically try `http://localhost:4180`; if unavailable, it keeps local sample data without interrupting the user.
- The Xcode-runnable iOS wrapper still exists as a legacy packaging reference, but current development, QA, and acceptance should target the browser webpage/PWA path unless the user explicitly asks to revive native packaging.
- Version 1 supports stock search without login for basic analysis.
- Login is required to save watchlists, preferences, reminders, and analysis history.
- Watchlist is required for personalized tracking; portfolio details are optional.
- User can select analysis style: conservative, balanced, or aggressive.
- The product is free at launch; future monetization is not part of MVP.
- Daily automatic log summary runs at 6:00 AM Australia/Brisbane time.
